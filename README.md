# Ahmad Taiwo — Frontend Engineer Portfolio

A modern, cinematic, and fully responsive personal portfolio built with Next.js, TypeScript, and Framer Motion. Designed with obsessive attention to UI detail and user experience.

🔗 **Live site:** [ahmadtaiwo.vercel.app](https://ahmadtaiwo.vercel.app)
📁 **Repository:** [github.com/Solataiwo-15/ahmadtaiwo](https://github.com/Solataiwo-15/ahmadtaiwo)

---

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 + CSS Variables
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Syne (display) · DM Sans (body) · DM Mono (mono)
- **Deployment:** Vercel

---

## Features

- 🌙 Dark / Light mode with localStorage persistence and zero flash
- 🎨 Aurora blob background with interactive cursor-reveal grid
- 🧲 Fixed social dock on desktop, inline icons on mobile/tablet
- ✨ Scroll-triggered entrance animations throughout
- 🔡 Scramble text effect on contact heading
- 🃏 Flip contact cards with copy-to-clipboard
- 🧲 Magnetic CTA button on contact section
- 🖼️ Project screenshot cards with hover overlays and live/GitHub links
- ⚡ Timeline experience section with education and strengths card
- 📱 Fully responsive — mobile, tablet, desktop
- 🚀 Deployed on Vercel with automatic CI/CD from GitHub

---

## Sections

- **Hero** — Full-viewport typographic name, status badge, CV download, stats
- **About** — Bio, profile card, floating badge
- **Skills** — Categorized tech stack with animated chips
- **Projects** — Screenshot bento grid with hover overlays (6 projects)
- **Experience** — Timeline with freelance, internships, education, and strengths
- **Contact** — Scramble heading, magnetic button, flip reveal cards, availability banner

## Getting Started

```bash
# Clone the repo
git clone https://github.com/Solataiwo-15/ahmadtaiwo.git

# Install dependencies
cd ahmadtaiwo
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Deployment

Connected to Vercel. Every push to `main` triggers an automatic deployment.
