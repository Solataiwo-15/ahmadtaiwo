# Ahmad Taiwo — Frontend Engineer Portfolio

A modern, animated, and fully responsive personal portfolio built with Next.js, TypeScript, and Framer Motion.

🔗 **Live site:** https://ahmadtaiwo.vercel.app/

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + CSS Variables
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Syne (display) + DM Sans (body) + DM Mono
- **Deployment:** Vercel

## Features

- 🌙 Dark / Light mode with localStorage persistence
- 🎨 Aurora blob background with interactive cursor grid
- ✨ Scroll-triggered animations throughout
- 📱 Fully responsive — mobile, tablet, desktop
- 🧲 Magnetic CTA button on contact section
- 🃏 Flip cards with copy-to-clipboard functionality
- 🔡 Scramble text effect on contact heading
- 💻 Animated code card in hero section
- 🖼️ Project screenshots with hover overlays
- ⚡ Timeline experience section

## Sections

- **Hero** — Name, role, code card, tech marquee
- **About** — Bio, profile card, mini stats
- **Skills** — Categorized tech stack with animated chips
- **Projects** — Screenshot cards with live + GitHub links
- **Experience** — Timeline with education and strengths
- **Contact** — Magnetic button, flip cards, availability banner

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deployed on Vercel. Push to `main` branch triggers automatic deployment.
